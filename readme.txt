Achieve the "draft" posts in some categories timing and quantitative release

== Description ==

Posts Release By Timing Plugin achieve the "draft" posts in some categories timing and quantitative release by setting release time and release amount/number.
it need used together with "WP Missed Schedule Fix Failed Future Posts" Plugin which decided the timing posts release amounts and time. 


== Development Blog ==

[ChenLei Wordpress Plugin Development Page](http://www.qchenlei.com/posts-release-by-timing-plugin)

== Installation ==

1. Upload `posts-release-by-timing.zip` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Option it in admin posts menu section

== Frequently Asked Questions ==
